¡Demostremos el modelo de caja de CSS!

El modelo de caja de CSS es esencialmente una caja que envuelve cada elemento HTML. Consiste en: bordes, relleno, márgenes y el contenido real.

Este texto es el contenido de la caja. Hemos añadido un ancho (widht) de 300px un alto (height) de 200px, un relleno (padding) de 20px, un margen (margin) de 30px y un borde azul solido de 10px.
